# ShopStream
### An End-to-End E-Commerce Data Engineering and Analytics Platform

A data engineering pipeline for e-commerce order data — EDA in Hive, ML model building in PySpark, and orchestration via Airflow with a scikit-learn model, deployed on AWS with a live dashboard.

*CDAC Thiruvananthapuram — PGCP Big Data Analytics, Final Project (Feb 2026 batch)*
